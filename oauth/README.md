Socialauth for phalcon
===========
Provides oAuth support in PHP 5.3+ and is very easy to integrate with any project which requires an oAuth client.

Service support
---------------
The library supports both oAuth 1.x and oAuth 2.0 compliant services. A list of currently implemented services can be found below. 

Included service implementations
--------------------------------
- OAuth1
    - BitBucket
    - Etsy
    - FitBit
    - Flickr
    - Tumblr
    - Twitter
- OAuth2
    - Amazon
    - BitLy
    - Box
    - Dailymotion
    - Dropbox
    - Facebook
    - Foursquare
    - GitHub
    - Google
    - Heroku
    - Instagram
    - LinkedIn
    - Microsoft
    - PayPal
    - RunKeeper
    - SoundCloud
    - Vkontakte
    - Yammer
    - Reddit
- more to come!
